  📋 SUNUCUDA YAPILACAKLAR

  1. Dosyaları yükledikten sonra:
  # Backend için
  cd backend
  npm install
  cp .env.example .env
  # .env dosyasını düzenle (MongoDB, JWT_SECRET vs)
  npm run build
  npm start

  # Frontend için
  cd frontend
  npm install
  cp .env.local.example .env.local
  # .env.local dosyasını düzenle (NEXT_PUBLIC_API_URL vs)
  npm run build
  npm start

  2. uploads klasörünü oluştur:
  mkdir -p backend/uploads/{products,banners,pages,albums,gallery}

  3. Process Manager kullanın (PM2 önerilir):
  npm install -g pm2
  pm2 start backend/dist/server.js --name ottoman-backend
  pm2 start "npm start" --name ottoman-frontend --cwd frontend/
  pm2 save
  pm2 startup

  Dosyaları transfer için rsync veya scp kullanabilirsiniz:
  rsync -av --exclude-from='.gitignore' ottoman-platform/ user@server:/path/to/app/
